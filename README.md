# Cloud Data Mining

## Setup


### Client

1. Go to client directory
   
```$ cd client```

2. Install dependencies 
   
```$ yarn install```

3. Build the client

```$ npm run build```


### Run server

1. From the root of the project, install dependencies

```$ npm ci```

2. start server

```$ npm run start```

3. Access the site at 
    https://localhost:3001/
